🐧 wayland-native

Voice to text,
from your terminal.

dictate records from PipeWire, transcribes via Mistral, Groq, or local Whisper, and pipes clean text to stdout. Built with Rust.

Install dictate View source →
# install
$ curl -fsSL https://dictate.adityamer.dev/install.sh | sh

# use
$ dictate --pipe-to wl-copy
$ pkill --signal SIGUSR1 dictate

# result
→ "Schedule a meeting for next Tuesday"
How it works

A single UNIX signal controls the flow. No daemon polling, no wasted resources.

Bind

Assign a keybind in your compositor

Speak

Records from PipeWire

Signal

SIGUSR1 transcribes

Text

Piped to stdout / wl-copy

Built for the terminal

A UNIX citizen. Composable. Zero overhead when idle.

⚡ Signal-driven

SIGUSR1 triggers transcription. No polling.

🔧 Pipe anywhere

--pipe-to wl-copy, ydotool, or any command.

🎵 Audio feedback

Beeps confirm recording events.

🌐 Multi-provider

Mistral, Groq, or local Whisper.

🏗️ Wayland native

PipeWire. Hyprland, Niri, GNOME, KDE.

🔒 Privacy

Local Whisper — audio stays on your machine.
