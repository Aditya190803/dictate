🐧 Wayland-native speech-to-text for Linux

Speak naturally.
Type instantly.

dictate is a signal-driven CLI that transcribes your voice using Mistral, Groq, or local Whisper — then pipes text to stdout. Built with Rust + PipeWire.

Get started View source →
# install + run
$ curl -fsSL https://dictate.adityamer.dev/install.sh | sh
$ dictate --pipe-to wl-copy
$ pkill --signal SIGUSR1 dictate

→ "Schedule a meeting for next Tuesday at 3 PM"
How it works

A single UNIX signal controls everything. No daemon polling.

1

Bind

Keybind in your compositor

2

Speak

Records from PipeWire

3

Signal

SIGUSR1 transcribes

4

Text

Piped to stdout/clipboard

Built for the terminal

UNIX composable. Zero overhead when idle.

⚡ Signal-driven

SIGUSR1 triggers transcription. No polling, no wasted cycles.

🔧 Pipe anywhere

--pipe-to wl-copy, ydotool, sed, or any command.

🎵 Audio feedback

Musical beeps confirm recording start, stop, success.

🌐 Multi-provider

Mistral, Groq, or local Whisper.

🏗️ Wayland native

PipeWire. Hyprland, Niri, GNOME, KDE, Sway.

🔒 Privacy

Local Whisper — audio never leaves your machine.
