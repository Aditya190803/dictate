dictate
How it works Features GitHub Install

Voice to text,
from your terminal.

A signal-driven CLI that records from PipeWire, transcribes via Mistral, Groq, or local Whisper, and pipes text to stdout.

🦀 Rust 🔊 PipeWire 🐧 Wayland
Install View source
# install
$ curl -fsSL https://dictate.adityamer.dev/install.sh | sh

# transcribe
$ dictate --pipe-to wl-copy
$ pkill --signal SIGUSR1 dictate

→ "Schedule a meeting for Tuesday 3 PM"
How it works

SIGUSR1 controls the flow. No daemon polling.

01

Bind

Keybind in compositor

02

Speak

PipeWire recording

03

Signal

SIGUSR1 transcribes

04

Text

stdout / clipboard

Features

Composable. Zero overhead when idle.

Signal-driven

SIGUSR1 triggers transcription. No polling.

Pipe anywhere

--pipe-to wl-copy, ydotool, or any command.

Audio feedback

Beeps confirm recording events.

Multi-provider

Mistral, Groq, or local Whisper.

Wayland native

PipeWire. Hyprland, Niri, KDE, GNOME.

Privacy

Local Whisper — audio stays on your machine.
