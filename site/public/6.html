Press a key. Speak.
Get text.

dictate is a signal-driven CLI that transcribes your voice via Mistral, Groq, or local Whisper and pipes text to stdout. Built with Rust + PipeWire.

Install dictate View source →
# install
$ curl -fsSL https://dictate.adityamer.dev/install.sh | sh

# use
$ dictate --pipe-to wl-copy
$ pkill --signal SIGUSR1 dictate

→ "Schedule a meeting for next Tuesday at 3 PM"
How it works

SIGUSR1 controls the entire flow. Zero polling.

01

Keybind in compositor

02

PipeWire recording

Signal

SIGUSR1 transcribes

04

stdout / clipboard

Features

UNIX composable. Zero overhead when idle.

Signal-driven

SIGUSR1 triggers transcription. No polling.

Pipe anywhere

--pipe-to wl-copy, ydotool, or any command.

Audio feedback

Beeps confirm recording events.

Multi-provider

Mistral, Groq, or local Whisper.

Wayland native

PipeWire. Hyprland, Niri, GNOME, KDE.

Privacy

Local Whisper — audio stays local.
